Tentang SIAP

SIAP (Sistem Informasi Agenda Pimpinan) adalah aplikasi berbasis web interaktif yang dirancang khusus untuk mempermudah pengelolaan, pemantauan, dan rekapan jadwal kegiatan pimpinan di lingkungan Badan Perencanaan Pembangunan Daerah (BAPPEDA) Kabupaten Bengkulu Tengah.

Database & Backend

Sistem Serverless memanfaatkan Google Sheets & Google Apps Script (GAS) sebagai Cloud Database & API.

Bot WA & Smart Fitur

Dilengkapi Reminder H-1 & 2 Jam, fitur deteksi bentrok jadwal, dan broadcast notulensi via API Sidobe.
